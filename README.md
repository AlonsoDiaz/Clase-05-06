# 🍽️ Proyecto Zomato - Análisis de Datos

## 📌 Descripción

Proyecto de análisis exploratorio de datos utilizando el dataset de Zomato, desarrollado colaborativamente mediante Git y GitHub para aplicar buenas prácticas de control de versiones y trabajo en equipo.

El proyecto busca identificar patrones relacionados con:

* ratings de restaurantes,
* precios,
* tipos de comida,
* servicios online,
* y comportamiento general de los datos.
